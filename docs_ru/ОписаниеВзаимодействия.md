# Вход в семью, или её cоздание


###### Начало использования
При первом взаимодействии с ботом пользователю будет предложено войти в существующую семью или создать новую, если он не является членом ни одной семьи. 
Для входа в существующую семью необходимо будет предоставить код-приглашение, который можно получить у любого пользователя, который уже состоит в данной семье. Код-приглашение можно получить с помощью команды `/get_invite_code` в телеграм боте. 
В любой момент можно присоединиться к другой семье, но это приведет к выходу из текущей семьи. Один телеграм-аккаунт – одна семья.
___

# Ведение учёта продуктов

#### Взаимодействие с приложением
Основное взаимодействие пользователя с базой данных происходит в интерфейсе Веб-приложения в Telegram. Пользователь сам вписывает названия продуктов, учёт которых хочет вести.
###### Возможности пользователя
Пользователь может:
- Добавить название продукта;
- Добавить описание продукта, например:
	- где продукт продаётся,
	- срок годности продукта;
- Добавить единицы измерения продуктов. Для каждого продукта они могут быть уникальны;
- У каждого продукта будет автоматически генерироваться иконка (эмодзи) на основе названия, заменяя собой фотографии для экономии места;
- Задать категории товаров. Категории должны быть предложены по названию товара;

###### Покупка
При покупке пользователь добавляет в систему количество продукта. Когда продукт заканчивается, следует убавлять его количество.
___

# Уведомления

Пользователь может сделать запрос на покупку любого продукта. Сообщение с запросом будет доставлено всем членам семьи в телеграм боте. 
Также пользователь может выбрать продукты, которые его интересуют, и подписаться на рассылку. В случае, если продукт закончится, то он автоматически получит уведомление об этом.
